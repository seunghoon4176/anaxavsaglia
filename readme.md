# 1vs1 격투게임 조작키 안내

## Player 1 (왼쪽)
- 이동: **A** (왼쪽), **D** (오른쪽)
- 점프: **W**
- 공격: **S**
- 방어: **Q** (누르고 있는 동안)
- 대시: **왼쪽 Shift (LSHIFT)**

## Player 2 (오른쪽)
- 이동: **←** (왼쪽), **→** (오른쪽)
- 점프: **↑**
- 공격: **↓**
- 방어: **오른쪽 Ctrl (RCTRL)** (누르고 있는 동안)
- 대시: **오른쪽 Shift (RSHIFT)**

> 공격과 점프는 키를 눌렀을 때 한 번만 동작합니다. 방어와 대시는 키를 누르고 있는 동안 유지됩니다.
